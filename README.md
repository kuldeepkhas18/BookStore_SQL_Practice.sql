# 📚 Book Store SQL Practice

This project contains SQL practice queries based on a Book Store Database using PostgreSQL.

## Database Tables

- Books
- Customers
- Orders

## Topics Covered

- SELECT
- WHERE
- ORDER BY
- GROUP BY
- HAVING
- JOIN
- Aggregate Functions
- DISTINCT
- LIMIT
- Foreign Keys

9. List all genres

13. Average Fantasy book price
14. Customers with at least two orders
15. Most frequently ordered book
16. Top 3 Fantasy books
